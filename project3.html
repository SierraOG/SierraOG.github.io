GPS Navigation Solution.

In the final laboratory for the Satellite-Based Navigation course, a navigation solution was derived in MATLAB based on pseudorange measurements that were calculated from observation data from an unknown continuously-operating reference station (CORS) receiver. The navigation solution implemented a least-squares techniques using QR decomposition to determine the location of the receiver. Pseudorange calculations accounted for ionospheric, tropospheric, earth rotation, and relativistic effects.

It can be shown from this that the navigation solution converges quickly on the actual location and has a large attractor region which gives leniancy to the initial position estimate provided.
Pseudorange calculations in this project accounted for ionospheric, tropospheric, earth rotation, and relativistic effects. This figure shows the average error of each effect for a 100 second period. From this figure it is clear that out of these four effects, the largest source of error, if not accounted for, would have been the tropospheric effects followed by the Earth rotation, relativistic effects, and finally the ionospheric effects.
