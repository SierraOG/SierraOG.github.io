Better Reads site.

This site has been taken down, however users could enter a description and search for books on the site. The description would be fed into a Natural Language Processing model which returned the top 10 closest matching books. The user could then read more information about each book, like it's description and rating, as well as open up the book in Amazon, Google Books, or Good Reads in a new tab. Once logged in the user could like books to add them to their saved books list and find them later.
